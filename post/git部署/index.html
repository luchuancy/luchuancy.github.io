git相关命令

1
2
3
4
5
6
7
8

$ cd public
$ git init
$ git remote add origin https://github.com/luchuancy/luchuancy.github.io.git
$ git status //查看本地自己修改了多少文件
$ git add .   //添加远程不存在的git文件
$ git commit -m "commit 211109"   //提交修改
$ git push -u origin master    //更新到远程服务器上
							//本地的master分支推送到origin主机，同时指定origin为默认主机，后面就可以不加任何参数使用git push了。

git 解除SSL验证

1

$ git config --global http.sslVerify "false"

git访问过慢

  1. 解析git ip地址 通过 https://www.ipaddress.com/ 获取 github.com 和 github.global.ssl.fastly.net 的ip地址，得到ip地址
1
2
3

// github的ip地址
github.com ...
github.global.ssl.fastly.net ...

  1. 添加到hosts文件

    打开文件：C:/windows/system32/drivers/etc/hosts，需要把上面获取的两个ip地址添加到系统hosts文件的末尾。

  2. 终端运行

1

$ ipconfig /flushdns

SSH key设置

  • 配置用户名和邮箱
1
2

$ git config --global user.name "用户名"
$ git config --global user.email  "邮箱"
  • 生成SSH密钥
1
2

$ ssh-keygen -t rsa -C "邮箱"
$ cat id_rsa.pub  // 文件内容即SSH密钥
  • github设置密钥
1

$ ssh -T git@github.com // 检查密钥是否设置成功
