在VSCode中搭建go环境

2021-11-10

Contents

  • 配置GOPATH环境变量
1

$ go env -w GOPATH=我们自己的工作区路径
  • 先打开Gomod，再配置七牛云代理
1
2

$ go env -w GO111MODULE=on
$ go env -w GOPROXY=https://goproxy.cn,direct
  • Go Modules的使用
1
2
3

$ cd $GOPATH/src/...    //main.go 的前一个文件夹
$ go mod init
$ go mod tidy // 安装依赖
